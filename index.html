01Basics

Random experiment

A random experiment has possible outcomes but the exact outcome is uncertain before it happens.

  • CBSE writing: Define the experiment and sample space before calculation.
  • JEE focus: Most mistakes disappear when outcomes are listed systematically.
  • Common trap: Do not count outcomes that are not equally likely.
02Sample

Sample space

The sample space S is the set of all possible outcomes.

  • CBSE writing: Write n(S) when using classical probability.
  • JEE focus: For two-stage experiments, use ordered pairs or tree branches.
  • Common trap: Missing one outcome changes the denominator.
03Event

Event and complement

An event A is a subset of sample space and its complement is everything outside A.

  • CBSE writing: Use P(not A)=1-P(A) when direct counting is long.
  • JEE focus: Complement method is powerful for at least one type questions.
  • Common trap: At least one is not the same as exactly one.
04Union

Union and intersection

Union means A or B, while intersection means A and B.

  • CBSE writing: Use the addition theorem with overlap correction.
  • JEE focus: Draw a Venn diagram before substituting.
  • Common trap: Do not add P(A)+P(B) without subtracting overlap.
05Conditional

Conditional probability

P(A|B) measures probability of A after B is known to have occurred.

  • CBSE writing: Write the condition clearly in the denominator.
  • JEE focus: Conditional probability changes the sample space.
  • Common trap: P(A|B) is usually not equal to P(B|A).
06Multiply

Multiplication rule

The chance of A and B is P(A) times P(B|A).

  • CBSE writing: Use it for dependent selections without replacement.
  • JEE focus: Tree diagrams are the fastest setup.
  • Common trap: Use P(B) only when events are independent.
07Independent

Independent events

Independent events do not affect each other, so P(A|B)=P(A).

  • CBSE writing: Show independence using P(A and B)=P(A)P(B).
  • JEE focus: Independence is a condition, not a feeling.
  • Common trap: Mutually exclusive events are not independent unless one event has probability zero.
08Bayes

Bayes theorem

Bayes theorem reverses conditional probability by weighting possible causes.

  • CBSE writing: Use total probability in the denominator.
  • JEE focus: Identify partitions first, then reverse the branch.
  • Common trap: Denominator must include all ways the observed event can happen.

Formula sheet

High-yield formulas with exact usage conditions.

Use these as quick revision cards before solving examples.

Classical

Classical probability

P(A) = n(A) / n(S)

Favourable equally likely outcomes divided by total equally likely outcomes.

  • Example: One die: P(even)=3/6=1/2.
  • Trap: Requires equally likely outcomes.
Range

Probability bounds

0 <= P(A) <= 1, P(S)=1, P(empty)=0

Probability is never negative or above one.

  • Example: A sure event has probability 1.
  • Trap: A probability like 7/5 signals an error.
Complement

Complement rule

P(not A) = 1 - P(A)

Everything outside A fills the rest of sample space.

  • Example: P(no head in 3 tosses)=1-P(at least one head).
  • Trap: Use complement carefully with wording.
Addition

Addition theorem

P(A union B) = P(A)+P(B)-P(A intersection B)

Overlap is counted twice, so subtract it once.

  • Example: P(A or B) with overlap.
  • Trap: For disjoint events, intersection is 0.
Conditional

Conditional probability

P(A|B) = P(A intersection B) / P(B), P(B)>0

Probability of A inside the reduced sample space B.

  • Example: Card is king given it is a face card.
  • Trap: Denominator is the condition.
Multiply

Multiplication rule

P(A intersection B) = P(A) P(B|A)

Useful for successive dependent events.

  • Example: Two cards without replacement.
  • Trap: Second probability may change after first event.
Independent

Independence test

P(A intersection B) = P(A)P(B)

Events are independent if product rule uses unchanged probability.

  • Example: Coin toss and die roll.
  • Trap: Disjoint does not mean independent.
Bayes

Bayes theorem

P(Ai|B)=P(Ai)P(B|Ai) / sum P(Ak)P(B|Ak)

Reverses a conditional probability across a partition.

  • Example: Choose a box after observing a defective item.
  • Trap: Include every partition case in denominator.

Derivation bank

Proofs that can be reproduced in exams.

Open a proof, follow the steps, and check the final result.

Derive complement rule
  1. A and not A have no common outcome.
  2. Together they form the whole sample space S.
  3. So P(A)+P(not A)=P(S).
  4. Since P(S)=1, P(not A)=1-P(A).

Final: P(not A)=1-P(A).

Shortcut: A and not A fill the sample space.

Warning: A and its complement must be disjoint.

Derive addition theorem
  1. Counting A plus counting B includes the common part twice.
  2. The common part is A intersection B.
  3. Subtract one copy of the common part.
  4. This gives probability of A union B.

Final: P(A union B)=P(A)+P(B)-P(A intersection B).

Shortcut: Subtract the overlap once.

Warning: For disjoint events the overlap term is zero.

Derive multiplication rule
  1. Use P(B|A)=P(A intersection B)/P(A).
  2. Multiply both sides by P(A).
  3. Get P(A intersection B)=P(A)P(B|A).
  4. This supports tree diagram branch multiplication.

Final: P(A intersection B)=P(A)P(B|A).

Shortcut: Rearrange conditional probability.

Warning: P(A) must be nonzero.

Derive Bayes theorem
  1. Start with P(Ai|B)=P(Ai intersection B)/P(B).
  2. Use multiplication rule: P(Ai intersection B)=P(Ai)P(B|Ai).
  3. Use total probability for P(B).
  4. Substitute to get Bayes theorem.

Final: P(Ai|B)=P(Ai)P(B|Ai)/sum P(Ak)P(B|Ak).

Shortcut: Use multiplication rule twice.

Warning: A1,A2,... must partition the sample space.

BasicsEasy

1. What is the probability of getting an even number on one die?

Answer: 1/2

Even outcomes are 2,4,6 out of 6 outcomes.

Trap: All die outcomes are equally likely.

SampleEasy

2. How many outcomes are there when two coins are tossed?

Answer: 4

HH, HT, TH, TT.

Trap: HT and TH are different ordered outcomes.

ComplementMedium

3. If P(A)=0.35, find P(not A).

Answer: 0.65

Use 1-P(A)=1-0.35.

Trap: Complement means outside A only.

AdditionMedium

4. If P(A)=0.5, P(B)=0.4, P(A and B)=0.2, find P(A or B).

Answer: 0.7

0.5+0.4-0.2=0.7.

Trap: Subtract overlap once.

ConditionalMedium

5. If P(A and B)=0.18 and P(B)=0.6, find P(A|B).

Answer: 0.3

P(A|B)=0.18/0.6.

Trap: Divide by probability of the condition B.

MultiplyMedium

6. Two cards are drawn without replacement. Why does the second probability change?

Answer: The sample space has one fewer card

Without replacement changes remaining outcomes.

Trap: Do not use independent multiplication.

IndependentMedium

7. If A and B are independent and P(A)=0.3, P(B)=0.5, find P(A and B).

Answer: 0.15

Use P(A)P(B)=0.3 x 0.5.

Trap: Only use this when independence is given or proved.

DisjointHard

8. Can two non-empty mutually exclusive events be independent?

Answer: No

P(A and B)=0 but P(A)P(B)>0 if both have positive probability.

Trap: Mutually exclusive is different from independent.

BayesHard

9. What is the denominator in Bayes theorem?

Answer: Total probability of the observed event

Add P(Ak)P(B|Ak) over all partition cases.

Trap: Do not include only one branch.

CBSEMedium

10. What must be written before applying classical probability?

Answer: Favourable outcomes and total equally likely outcomes

Use n(A) and n(S).

Trap: Do not skip sample space in board solutions.

JEEHard

11. Why is a tree diagram useful for probability?

Answer: It organizes successive conditional branches

Multiply along branches and add suitable paths.

Trap: Branch probabilities must match previous outcomes.

TrapMedium

12. Is P(A|B) always equal to P(B|A)?

Answer: No

They use different conditions and denominators.

Trap: Only special cases make them equal.

Core formulas

  • Classical probability: P(A) = n(A) / n(S)
  • Probability bounds: 0 <= P(A) <= 1, P(S)=1, P(empty)=0
  • Complement rule: P(not A) = 1 - P(A)
  • Addition theorem: P(A union B) = P(A)+P(B)-P(A intersection B)
  • Conditional probability: P(A|B) = P(A intersection B) / P(B), P(B)>0
  • Multiplication rule: P(A intersection B) = P(A) P(B|A)

Common traps

  • Classical probability needs equally likely outcomes.
  • At least one often becomes easier by complement.
  • Union needs overlap correction.
  • Conditional probability changes the sample space.
  • Independent and mutually exclusive are different ideas.
  • Bayes denominator must include every possible cause.
